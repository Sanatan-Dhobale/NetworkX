# 🌐 NetworkX

A modern **Full Stack Professional Networking Platform** inspired by LinkedIn, built using the **MERN Stack**. NetworkX allows users to build professional connections, share posts, upload resumes, and download their complete profile as a professionally formatted PDF resume.

---

## 🚀 Features

### 👤 Authentication
- User Registration
- Secure Login
- JWT Authentication
- Protected Routes

### 👥 User Profile
- Edit Profile Information
- Upload Profile Picture
- View Other User Profiles
- Download Profile as PDF Resume

### 🤝 Professional Network
- Send Connection Requests
- Accept Connection Requests
- View Connected Users
- Professional Network Dashboard

### 📝 Posts
- Create Text Posts
- Upload Image Posts
- Like Posts
- Share Posts
- Comment on Posts
- View Feed

### 📄 Resume Generator
- Automatically generates a professional PDF resume from the user's profile information.

- MongoDB

- JSON Web Token (JWT)
